About the Jam

Join the 2025 Vibe Coding Game Jam, where AI powers the future of gaming. Create free, web-accessible, instantly playable games using at least 80% AI-generated code. Deadline: April 1, 2025, at 23:59 UTC.

  1. At least 80% of code must be written by AI.
  2. Game must be accessible on the web without any login or signup and free-to-play (preferably its own domain or subdomain).
  3. Game does not need to be multiplayer, but single-player is fine; multiplayer is a bonus.
  4. Can use any engine, but Three.js is recommended.
  5. No loading screens or heavy downloadsâ€”games must load almost instantly (except for an optional username prompt).
